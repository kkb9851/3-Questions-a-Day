### 작동 방식 예시

입력 배열: `[2, 7, 3]`

#### 1회전

* `a=2`, `b=7`
* `LCM(2,7) = (2×7) // gcd(2,7) = 14`
* `arr = [3, 14]`

#### 2회전

* `a=3`, `b=14`
* `LCM(3,14) = (3×14) // gcd(3,14) = 42`
* `arr = [42]` → 종료

최종 결과: `42`
